#### System Requirements

#### Getting Started

1. Clone the repository.

    ```shell
    git clone https://github.com/BlairLi/CISC498_Web-based_Real-time_Explorer.git
    cd CISC498_Web-based_Real-time_Explorer
    ```
1. Install all dependencies with NPM.

    ```shell
    npm install
    ```
1. Run the frontend server.

    ```shell
    npm run start
    ```
